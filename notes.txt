path module
path.basename()
returns the last portion of a path similar to the unix basename command;

keywords
in node.js these keywords are special variable available in commonjs module that provide the directory name and file name of the current module.
__dirname ->current directory name
__filename ->current filename

path.join(__dirname,filename)
Joins all given path segments together using the platform-specific seperator as a delimiter then normalizes the result 

path.extname()
returns the extension of a path, from the last occurance of the .(period) character to the end of the string

path.resolve()
resolves a sequence of paths or path segments into an absolute path,processing from right to left until an absolute path is considered

path.parse()
returns an object whose properties prepresent significant elements of the path
